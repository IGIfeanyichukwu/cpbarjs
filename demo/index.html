Cpbar.js
...library for circular progress bars

About

Cpbar.js is a light-weight javascript library that provides you a seamless way of creating circular progress bars. All you need to do is call a function, provide preference and cpbar.js will take over from there.

Docs

Installation

1. Using CDN

<script src="cdn.example.com/cpbar.js"></script>

2. Use Package Manager

npm i cpbarjs

3. Importing Cpbar.js

If you are using imports, use the following to import Cpbar.js

import CPB from 'cpbar'

Initializing

To create the circular progress bar, call the draw method and pass in an object of your preference for the circular progress bar.

CPB.draw({ elementClass: "example" })

The above will create a circular progress bar with default preference.

Always provide the elementClass value so cpbar.js will know where to render the progress bar to.

Default Circular Progress Bar

CPB.draw({ elementClass: "example" }).

Properties Function Default Value Possible Values
barSize changes progress bar size 200 0 - 1000
barBgColor changes background color grey any browser color
trackColor changes track color #eee any browser color
indicatorColor changes indicator color black any browser color
valueColor changes background color black any browser color
barBgOpacity changes background opacity 0 any browser color
